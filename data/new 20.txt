Selesai! Saya telah membuatkan aplikasi Toko Tiket Online Hollynice Bureau lengkap dengan fitur Frontend (Desktop & Mobile terpisah), Backend Admin, Sistem Tiket QR Code, dan POS Scanner.

Aplikasi ini dibangun menggunakan PHP Native (struktur ringan dan cepat) dan Tailwind CSS (modern UI), siap dijalankan di XAMPP.

### 1. Fitur Utama
- Tampilan Modern : Desain responsif dan menarik sesuai permintaan.
- Mobile Experience : Tampilan khusus (seperti aplikasi native) jika dibuka dari HP.
- Multi-Bahasa : Fitur ganti bahasa (Inggris / Indonesia).
- Manajemen Tiket : Tiket Masuk, Tur, dan Event.
- Sistem QR Code : Generate tiket QR otomatis setelah checkout.
- POS Scanner : Fitur untuk Operator men-scan dan memvalidasi tiket di lokasi.
- Simulasi Pembayaran : Siap untuk integrasi QRIS (saat ini simulasi sukses bayar).
### 2. Struktur Project
Aplikasi tersimpan di d:\xampp\htdocs\jan .

- index.php : Router utama.
- views/ : Folder tampilan (terpisah folder desktop , mobile , admin ).
- controllers/ : Logika pemrosesan data.
- config/ : Koneksi database.
- img/ : Menggunakan gambar wisata yang sudah tersedia.
### 3. Cara Menggunakan
1. Database : Saya sudah menjalankan skrip instalasi otomatis. Database hollynice_ticket_db beserta tabel dan data dummy sudah siap.
2. Akses Web : Buka browser dan akses http://localhost/jan/
3. Akun Login Default :
   - Admin : admin / admin123 (Akses Dashboard & Laporan)
   - Operator : operator / operator123 (Akses POS Scanner)
   - User : user / user123 (Untuk tes pembelian tiket)
### 4. Alur Tes Aplikasi
1. Beli Tiket (Sebagai User) :
   - Login sebagai user .
   - Pilih produk (Tur Dieng, Tiket Kawah, dll).
   - Masuk ke Cart -> Checkout.
   - Lakukan simulasi pembayaran QRIS -> Klik "Saya sudah bayar".
   - Tiket dengan QR Code akan muncul.
2. Validasi Tiket (Sebagai Operator) :
   - Buka browser lain (atau Incognito).
   - Login sebagai operator .
   - Masuk ke menu "POS Scanner".
   - Scan QR Code tiket yang tadi dibeli (bisa gunakan kamera laptop atau input manual jika perlu debug).
   - Sistem akan menampilkan "Access Granted" jika valid.
